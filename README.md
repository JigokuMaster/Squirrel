Squirrel is a simple application to create and scan QR codes.
The scanner is powered by ZBar library supporting bar codes as well as various QR code types like those with rounded corners.

# Supported platforms

* Symbian S60v3, S60v5, S^3 

# Requirements 

P.I.P.S v1.7 for S60v3 and S60v5

# Installation

* Download the binaries from [releases page](https://github.com/JigokuMaster/squirrel/releases) 

# Features

* Generate QR code from text, contact, notes, WIFI IP
* Generate QR code with specific version, colors ...etc
* Send/Save QR code image file. 
* Decode QR code or bar code from image file.
* Scan directly using camera.
* View/Copy/Send decoded content.
* Multilingual, available in six languages.
